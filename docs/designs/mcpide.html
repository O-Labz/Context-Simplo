VS Code

Extension Configuration

Install the MCP Client extension from the marketplace. Then, add the following to your settings.json:

"mcp.servers": {
  "simplo-mcp": {
    "command": "npx",
    "args": [
      "-y",
      "@simplo/mcp-server"
    ],
    "env": {
      "SIMPLO_API_KEY": "your_api_key_here"
    }
  }
}

Claude Code

CLI Tooling

Initialize Simplo directly within the Claude CLI environment using the setup command:

claude mcp add simplo

PRO TIP

Ensure your environment variables are set in your .zshrc or .bashrc for persistent CLI access.
